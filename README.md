# 🛒 ShopEase - E-Commerce Website

ShopEase is a full-stack E-Commerce Website developed using **PHP**, **MySQL**, **HTML**, **CSS**, **Bootstrap**, and **JavaScript**. It provides a complete online shopping experience for customers and an admin dashboard to manage products, categories, users, and orders.

---

# 📌 Features

## Customer Module

- User Registration
- User Login & Logout
- Home Page with Hero Slider
- Product Categories
- Product Search
- Product Details
- Add to Cart
- Wishlist
- Checkout
- Order Confirmation
- My Orders
- Order Details
- User Profile
- Product Reviews

# 🚀 Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into:

```
xampp/htdocs/
```

4. Open phpMyAdmin.
5. Create a database named:

```
ecommerce
```

6. Import the **ecommerce.sql** file.
7. Open your browser and visit:

```
http://localhost/ecommerce/
```
