# 🛒 ShopEase - E-Commerce Website

ShopEase is a full-stack E-Commerce Website developed using PHP, MySQL, HTML, CSS, Bootstrap, and JavaScript. The application allows customers to browse products, add items to the cart and wishlist, place orders, and track their purchases. It also includes a dedicated Admin Panel to manage products, categories, users, orders, and reports.

ecommerce

---

## 🚀 Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into:

```text
xampp/htdocs/
```

4. Open phpMyAdmin.
5. Create a database named:

```text
ecommerce
```

6. Import `ecommerce.sql`.
7. Open the browser and visit:

```text
http://localhost/ecommerce/
```
